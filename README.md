## 📋 Описание

**DOMIKTUT** - это современный веб-сайт для аренды коттеджей и домов. Платформа предоставляет удобный поиск, фильтрацию и бронирование жилья для различных мероприятий и отдыха.

## ✨ Основные возможности

- 🔍 **Умный поиск** - быстрый поиск домов по параметрам
- 🏠 **Каталог жилья** - большой выбор коттеджей и домов
- 🎯 **Фильтрация** - фильтрация по цене, количеству гостей, удобствам
- 📅 **Бронирование** - удобная система бронирования
- 📱 **Адаптивный дизайн** - корректное отображение на всех устройствах
- 🌐 **Мультиязычность** - поддержка русского, белорусского и английского языков
- 🎨 **Темы оформления** - светлая и тёмная темы
- ♿ **Доступность** - настройки размера шрифта, цветовой схемы

## 🛠️ Технологии

- **HTML5** - семантическая разметка
- **CSS3** - современные стили с использованием CSS Variables
- **JavaScript (ES6+)** - интерактивность и динамика
- **JSON Server** - имитация backend API
- **Figma** - дизайн и прототипирование

## 📦 Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone https://github.com/12programmer12/coursework.git
```

2. Перейдите в директорию проекта:
```bash
cd coursework
```

3. Откройте `index.html` в браузере или используйте Live Server

## 🌟 Функциональные возможности

### Для пользователей:
- Просмотр каталога домов с фотографиями
- Поиск по названию и местоположению
- Фильтрация по цене, количеству гостей, спальням
- Добавление в избранное
- Оформление заявки на подбор жилья
- Бронирование дат
- Дополнительные услуги (кейтеринг, трансфер и др.)

### Настройки доступности:
- Изменение размера шрифта (маленький, средний, большой)
- Выбор цветовой схемы (стандарт, тёплая, холодная)
- Включение/отключение изображений
- Светлая/тёмная тема

## 📱 Адаптивность

Сайт полностью адаптирован для:
- 📱 Мобильных устройств (320px+)
- 📱 Планшетов (768px+)
- 💻 Десктопов (1024px+)

## 🎯 Особенности реализации

- **Модульная архитектура** JavaScript кода
- **LocalStorage** для сохранения настроек пользователя
- **Intersection Observer** для ленивой загрузки изображений
- **CSS Grid** и **Flexbox** для современной вёрстки
- **CSS Variables** для удобной кастомизации тем
- **Semantic HTML** для лучшей доступности и SEO
